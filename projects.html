Примеры работ

Чат-приложение на React и ASP.NET Core

Реализация онлайн-чата с аутентификацией, отображением реальных пользователей и приватными беседами. Для хранения данных применялся PostgreSQL, подключение через SignalR и JWT авторизация.

Интерактивная галерея на JavaScript

Веб-галерея с фильтрацией, масштабированием, просмотром изображений в модальном окне. Использована адаптивная верстка, плавные CSS-анимации и оптимизация загрузки картинок.

REST API для школьного портала

Разработка backend на ASP.NET Core: регистрация, лента событий, работа с ролями пользователей, интеграция с внешними сервисами (email-рассылка, пуш-уведомления).

Сайт-визитка на HTML и CSS

Адаптивный одностраничный сайт с разделами “О себе”, “Навыки”, “Портфолио”. Использованы Google Fonts, flex/grid layout, современные CSS-переходы.
